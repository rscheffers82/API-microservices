  • Timestamp Microservice API
User stories:
  1. I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: December 6, 1982)
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or Unix timestamp, it returns null for those properties.

Example usage:

    http://rs82.herokuapp.com/timestamp/December 6, 1982
    http://rs82.herokuapp.com/timestamp/408005400

Example output:

  {
    "unix": 408005400,
    "natural": "December 6, 1982"
  }
