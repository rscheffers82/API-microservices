API Project Collection

This collection of 5 projects is part of Free Code Camp's curriculum, API section. The back-end is built on Node and Express and several incorporate MongoDB when a required. Some projects use additional libraries e.g. Moment.js and Mongoose Shortid-nodeps. The front-end of Exercise tracker is built using HTML, CSS, JS and Materialize CSS.
  • Exercise Tracker API

    Exercise Tracker allows users to register and log exercises. Once in the system exercises can be searched by start and finish date, or none to display all results. Creating a new user or exercise posts data to the back-end powered by Node, Express and MongoDB. Log data is provided via GET.

    POST routes display JSON. In contrast, front-end logic is added the GET logs route to display the logs below the three cards.

    Demo Code
  • URL Shortener API

    a URL Shortener API that shortens the provided URL and returns JSON data containing both the original and shortened link. The back-end is built on Node, Express and MongoDB.

    Demo Code
  • File Metadata API

    Through a form users can submit a file, which when posted, returns the filename and size (in bytes) in JSON format. The back-end is built on Node and Express. By utilizing Multer middleware, any file which is uploaded with a request adds a file and body object to the request object. Data from these objects makes up to the JSON response.

    Demo Code
  • Timestamp Microservice API

    This API takes a Unix timestamp or date in string format and returns both the Unix timestamp and the natural language form of that date. When no correct input is provided, null for both properties is returned. The back-end is built on Node and Express and the Moment.js library is used to format the dates.

    Demo Code
  • Request Header Parser API

    This API provides the user with their IP address, language and operating system for their browser. The back-end is built on Node and Express, and the Accepts library is used to help obtain and format the language of the user's system.
