API Project Collection

This collection of project is part of Free Code Camp's curriculum, API section. All back-end code is build on Node and Express and several incorporate MongoDB when a database is required. Depending on the project additional libraries are using e.g. Moment.js and Mongoose Shortid-nodeps. The front-end part of the Exercise tracker is build using HTML, CSS, JS and the Materialize CSS library.
