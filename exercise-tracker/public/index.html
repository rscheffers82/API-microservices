  • Welcome to Exercise Tracker! This app is part of FreeCodeCamp's curriculum API section.

    This app will let you create a new user, add exercises and view the logs. By creating a new user and adding exercises POST requests are made to the server. Log data is provided via a GET request. Give it a try by clicking one of the links below to get started.
